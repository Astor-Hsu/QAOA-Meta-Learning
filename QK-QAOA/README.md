1. Create a new conda environment with Python 3.10.18:
   ```bash
   conda create --name [name] python=3.10
   ```
2. Activate the newly created environment:
   ```bash
   conda create [name] 
   ```
3. Install the required package using pip:
   ```bash
   pip install -r requirements.txt
   ```
